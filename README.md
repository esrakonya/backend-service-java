# Java Backend Service

A professional-grade backend service built to demonstrate core software engineering principles, environment-based configuration, and structured logging.

## Tech Stack
- **Language:** Java 17
  - **Framework:** Spring Boot 3.2.5
  - **Build Tool:** Maven
  - **Logging:** SLF4J + Logback
  - **Version Control:** Git
  
## Project Status: Phase 1 (Core Infrastructure)
Currently, the project focuses on setting up a robust foundation:
- Integrated **Spring Boot 3.2.5** for enterprise-level capabilities.
- Configured **SLF4J/Logback** for structured and environment-aware logging.
- Established a clean **Maven** build lifecycle.
- Implemented **Environment Variable** management for flexible deployments.

## Getting Started

### Prerequisites
- JDK 17
- Maven 3.x

### How to Run
To run the application locally, use the following command:
```bash
mvn clean spring-boot:run
